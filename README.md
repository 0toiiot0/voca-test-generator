# 영어 단어 시험지 생성기

영어 단어를 등록하고 관리하며, 랜덤 순서의 시험지를 생성하여 PDF로 다운로드할 수 있는 웹 애플리케이션입니다.

## 주요 기능

### 1. 단어 관리 (CRUD)
- **추가**: 영어 단어와 한국어 뜻 입력
- **조회**: 등록된 모든 단어 목록 표시
- **수정**: 인라인 편집으로 단어 수정
- **삭제**: 확인 후 단어 삭제

### 2. 암기 상태 관리
- 체크박스로 암기 완료/미완료 표시
- 암기 완료된 단어는 취소선 스타일 적용
- 상태 변경 시 즉시 저장

### 3. 필터링
- **카테고리 필터**: 전체 / 완벽하게 외운 단어 / 헷갈리는 단어
- **날짜 필터**: 단어가 추가된 날짜별 필터링
- **복합 필터링**: 카테고리와 날짜 동시 적용

### 4. 시험지 생성
- Fisher-Yates 알고리즘으로 랜덤 섞기
- 매번 다른 순서로 생성
- 인쇄 친화적인 테이블 레이아웃

### 5. PDF 다운로드
- A4 사이즈, 고해상도 (300 DPI)
- 파일명 형식: `영어단어시험지_YYYYMMDD_HHMMSS.pdf`
- 로딩 인디케이터 표시

### 6. 사용자 경험
- 다크/라이트 테마 지원 (시스템 설정 자동 감지)
- 토스트 알림으로 액션 피드백
- 반응형 디자인 (모바일/태블릿/데스크톱)

## 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: CSS 변수 기반 테마 시스템, Flexbox/Grid 레이아웃
- **JavaScript (ES6+)**: Vanilla JS, 모듈화된 함수 구조
- **LocalStorage**: 클라이언트 사이드 데이터 저장
- **html2pdf.js**: PDF 생성 라이브러리

## 로컬 실행 방법

### 방법 1: Python 서버
```bash
# Python 3.x
python -m http.server 8000

# 브라우저에서 접속
# http://localhost:8000
```

### 방법 2: VS Code Live Server
1. VS Code에서 프로젝트 폴더 열기
2. Live Server 확장 설치
3. `index.html` 파일에서 우클릭 → "Open with Live Server"

### 방법 3: 직접 열기
`index.html` 파일을 브라우저에서 직접 열어도 동작합니다.
(단, 일부 브라우저에서 localStorage 기능이 제한될 수 있음)

## 사용 방법

### 1. 단어 추가
1. "단어 추가" 섹션에서 영어 단어와 한국어 뜻 입력
2. "단어 추가" 버튼 클릭

### 2. 단어 관리
- **수정**: 단어 항목의 "수정" 버튼 클릭 → 내용 수정 → "저장"
- **삭제**: "삭제" 버튼 클릭 → 확인
- **암기 완료**: 체크박스 클릭

### 3. 필터링
- 상단 탭에서 카테고리 선택 (전체/외운 단어/헷갈리는 단어)
- 드롭다운에서 날짜 선택
- "전체 보기" 버튼으로 필터 초기화

### 4. 시험지 생성 및 다운로드
1. "시험지 생성" 버튼 클릭
2. 미리보기 확인
3. "PDF 다운로드" 또는 "인쇄하기" 버튼 클릭

## 데이터 구조

```javascript
// 단어 객체
{
  id: string,        // 고유 ID (타임스탬프 기반)
  english: string,   // 영어 단어
  korean: string,    // 한국어 뜻
  mastered: boolean, // 암기 완료 여부
  dateAdded: string  // 추가 날짜 (ISO 8601)
}
```
